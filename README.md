Testing
-------

First start a local ElasticSearch:

```bash
$ docker run -p 9200:9200 -p 9300:9300 -e "discovery.type=single-node" docker.elastic.co/elasticsearch/elasticsearch:6.1.3
```

Run Common Test:

```bash
$ make ct
```
